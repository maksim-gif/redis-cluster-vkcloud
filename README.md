# Redis Cluster с Sentinel на VK Cloud

## Описание проекта
Развертывание отказоустойчивого Redis кластера с автоматическим фейловером на платформе VK Cloud с использованием Terraform.

## Архитектура
- **3 виртуальные машины** в разных зонах доступности
- **Redis Sentinel** для автоматического фейловера
- **Репликация 1 мастер + 2 реплики**

## Файлы проекта

### Terraform конфигурации
- `main.tf` - основные ресурсы ВМ
- `network.tf` - настройки сети
- `secgroup.tf` - группы безопасности
- `variables.tf` - переменные
- `vkcs_provider.tf` - провайдер VK Cloud
